SceneRotator HeadTracker
Version 0.1

This folder contains the lightweight head-tracking application used to control
the IEM SceneRotator via OSC for the supplementary listening package.

Quick Start:
1. Open `SceneRotatorHeadTracker.app`.
2. Select the desired camera.
3. Confirm that SceneRotator is listening on port `7000`.
4. Press `Start`.
5. Press `Calibrate` while looking straight ahead.

Notes:
- If macOS asks for camera permission, allow access.
- Use headphones for the listening demonstration.
- If tracking seems inactive, verify the OSC receive settings in SceneRotator.
