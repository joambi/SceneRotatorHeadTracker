# Paper Text Draft

## Supplementary Material With GitHub Link

A supplementary listening package accompanies this paper. In addition to a `REAPER` session containing the original Ambisonics material and the corresponding `IEM`-based binaural rendering chain, the package includes `SceneRotator HeadTracker`, a lightweight webcam-based utility for real-time control of the `IEM SceneRotator` via OSC. The software is provided through a public GitHub repository (`https://github.com/joambi/SceneRotatorHeadTracker`) together with a macOS application build, allowing reviewers and readers to reproduce the head-tracked listening condition described in this paper with minimal setup effort.

## Slightly More Reviewer-Oriented Version

To make the listening situation discussed in this paper easier to evaluate in practice, a supplementary package is provided. It contains a `REAPER` session with the original Ambisonics material, the relevant `IEM` plug-in configuration for binaural rendering, and `SceneRotator HeadTracker`, a lightweight webcam-based utility for controlling the `IEM SceneRotator` in real time. The software is available via GitHub (`https://github.com/joambi/SceneRotatorHeadTracker`) together with a macOS application build, so that reviewers and readers can directly compare static binaural reproduction and head-tracked binaural reproduction using the same source material discussed in the paper.

## Short Version

The supplementary material includes a `REAPER` session, the original Ambisonics source material, and a lightweight webcam-based head-tracking utility for real-time `IEM SceneRotator` control. The software is provided through GitHub as a macOS application build and is intended to support direct comparison between static and head-tracked binaural reproduction.
