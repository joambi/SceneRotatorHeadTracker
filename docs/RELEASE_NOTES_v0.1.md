# SceneRotator HeadTracker v0.1

First public research release of `SceneRotator HeadTracker`.

## Included

- Native macOS application build
- Webcam-based head tracking for `IEM SceneRotator`
- Direct OSC output to `/SceneRotator/ypr`
- `Yaw only` and `Full YPR` modes
- `Calibrate` and `Recenter`
- Repository documentation for build and research use

## Intended Use

This release is intended for supplementary listening material, reviewer evaluation, and practical reproduction of head-tracked binaural Ambisonics playback in `REAPER`.

## Requirements

- `macOS`
- `REAPER`
- `IEM Plug-in Suite`
- webcam
- headphones

## Notes

- Camera permission may need to be granted in macOS on first launch.
- The current release is macOS-only.
- `REAPER` and the `IEM Plug-in Suite` are external dependencies and are not bundled.
